# 制造商支持的自动驾驶仪

制造商支持的自动驾驶仪由板卡制造商维护和支持（制造商承诺在官方发布后四个月内确保与当前稳定版PX4版本的兼容性）。

:::tip
关于PX4项目自动驾驶仪板卡支持级别的更多信息，请参见：[px4.io/autopilots/](https://px4.io/autopilots/)。
:::

::: info
此类别包含未完全符合pixhawk标准的板卡；制造商会在不兼容的领域对板卡进行支持和维护。
:::
