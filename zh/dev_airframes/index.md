# 机体框架

PX4 具有灵活的 [控制分配系统](../concept/control_allocation.md)，通过统一代码库支持几乎任何可想象的机体类型/框架：

- **固定翼飞机**：常规飞机、飞翼布局、倒V尾翼飞机等。
- **多旋翼**：直升机、三旋翼、四旋翼、六旋翼、十二旋翼等不同几何构型。
- **垂直起降机体**：包含以下配置：尾座式、倾转旋翼以及QuadPlanes（飞机+四旋翼）。
- **无人地面车辆（UGVs/漫游车）**：已添加基础支持，实现手动和任务模式控制。

所有支持的框架类型和电机输出列表请参阅 [机体框架参考](../airframes/airframe_reference.md)。

本节为希望为 PX4 添加新机体或机体类型支持的开发者提供相关信息，包括仍在开发中的机体的构建日志。

::: info
PX4 也非常适用于其他车辆类型和通用机器人，包括潜艇、船只、两栖车辆，以及实验性飞机和火箭。
如有新的机体或框架类型需要支持，请告知我们。
:::

部分支持的机体框架构建日志可在 [机体/车辆构建](../airframes/index.md) 中找到。
:::
