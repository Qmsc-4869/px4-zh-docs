# 飞机（固定翼）

PX4 支持多种飞机几何结构，包括常规飞机、飞翼布局、倒V尾布局飞机等。

::: tip
通用配置可在 [机架参考 > 飞机](../airframes/airframe_reference.md#plane) 中查看。
这些配置可在配置过程中进行自定义。
:::
