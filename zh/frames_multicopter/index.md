# 多旋翼

PX4 支持多种单旋翼和多旋翼机型，包括直升机、三旋翼、四旋翼、六旋翼、十二旋翼等，涵盖多种几何结构（+型、宽型、X型等）。

::: tip
通用配置可参考 [机架参考 > 旋翼](../airframes/airframe_reference.md#copter)。
这些配置可在设置过程中自定义。
:::
